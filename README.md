# pymongo-api

## Как запустить

Запускаем mongodb и приложение

```shell
cd sharding-repl-cache && docker compose up -d && cd -
```

Инициализируем монго и заполняем данными

```shell
cd sharding-repl-cache && sh init.sh && cd -
```

## Как проверить

Откройте в браузере http://localhost:8080
